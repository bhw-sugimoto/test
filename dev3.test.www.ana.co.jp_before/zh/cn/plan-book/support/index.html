国际航班预订介绍

从预订机票到到达目的地，全程将您所搭乘航班的最新信息及时通知您，为您提供客服支持。除了浏览ANA网站之外，还可以通过电子邮件、SMS短信或ANA APP接收旅程状况的实时更新。在这选项卡中，任一方式都可以查看到便利实用的信息。

请注意：据报告Gmail会出现显示错误的航班信息（时间和航班号）的情况。关于最新的正确信息，请点击电子邮件的URL，在“确认预订内容”页面进行确认。
请登记在目的地也可使用的邮箱地址。

  • 登录邮箱地址

电子邮件通知

  • 如果搭乘航班的运行计划发生变更，我们会立即通过邮件通知您。如果您有已预订的航班，请登录电子邮箱。
  • 您的家人和朋友也可以收到邮件通知。关于通过各类电子邮件发送的信息，请参阅以下说明。
  • 电子邮件主要发送预订（包括航班变更、取消）及购票的相关信息。
  • 电子邮件会发送到作为主要联系人登记的电子邮箱中。仅可登记1个电子邮箱。

请注意：

  • 受安全软件和电子邮件的设置影响，来自ANA的邮件可能会被自动识别为垃圾邮件、广告邮件的文件夹或被删除。

    请提前将您使用的电子邮件系统设置为可接收来自”121.ana.co.jp“，”mail.ana.co.jp“或”amadeus.com“的邮件。

  • 值机的通知邮件来自”amadeus.com“。
  • 关于航班变更及座位变更的相关信息，通知对象仅限ANA承运的国际航班。
  • 在日本的网站购票时，邮件中附有行程单。
发送给旅客代表
  • 通知内容：主要发送有关预订（包括变更和取消）和购票相关的邮件。*1
  • 收信地址：预订时作为“旅客代表邮件地址”登录的收信人地址（每个预订可登录1个地址。）
  • 其他：在ANA日本网站购票时，邮件中附有收据。
  • *1 机型变更及机型变更导致的座位变更的通知，仅适用于ANA承运的国际航线航班。
乘客
  • 电子邮件主要发送有关值机和搭乘航班的信息。
    • 请注意：仅通知ANA承运的国际航班的航班变更及座位变更的相关信息。
  • 电子邮件将发送到预订时登记的“主要有关乘机、航班动态的电子邮件发送地址”。（每位乘机人可以登记一个单独的邮件地址。）
  • 通知邮件示例：
    • 附有电子客票旅程、行程单的邮件。
    • 出发24小时之前的值机通知（日本国内航班、其他航空公司航班、其他航空公司承运的代码共享航班除外）。
    • 出发前一天或当天的搭乘航班延误通知（日本国内航班、其他航空公司航班、其他航空公司承运的代码共享航班除外）。
    • 出发前一天或当天的搭乘航班取消通知（其他航空公司航班、其他航空公司承运的代码共享航班除外）。
朋友、家人、司机
  • 向乘客以外的人员发送ANA承运国际航班的出发和到达信息。
    • 请注意：日本国内航班、其他航空公司航班、其他航空公司承运的代码共享航班不会发送电子邮件。
  • 在机票出票后，会将电子邮件发送至为了接收出发和到达邮件而登记的电子邮箱。可登记2个电子邮箱。
  • 电子邮件发送时间如下。
    • 出发时
    • 到达2小时前（飞行时间在5小时以上的航班）
    • 到达时
希望升舱的乘客
  • 将通过电子邮件发送申请升舱的预订相关信息（包括航班变更和取消）。
    • 请注意：日本国内航班、其他航空公司航班、其他航空公司承运的代码共享航班不会发送电子邮件。
  • 电子邮件会发送到申请升舱时登记的电子邮箱中。1个预订仅可登记1个电子邮箱。

ANA APP的通知

  • 航班延误或取消的信息，将通过ANA APP的推送通知提醒您。
  • 请注意：本服务不可用于其他航空公司航班、其他航空公司承运的代码共享航班。
  • 将通知发送到已设定好接收ANA APP推送通知的设备。
    • 只有完成预订航班的ANA里程俱乐部成员才能收到此通知。

SMS短信通知

  • 登记电话号码后，将发送航班延误或取消的SMS短信（出发前一天或出发日）。
    • 请注意：SMS不可用于其他航空公司航班、其他航空公司承运的代码共享航班。
  • 如下国家无法通过SMS发送运航状况（有变更）。

    • 朝鲜民主主义人民共和国
    • 瑙鲁共和国
    • 北马其顿共和国
    • 梵蒂冈城国
    • 图瓦卢
    • 瓦利斯和富图纳

国际航班的日历提醒

从ANA网站或预订确认画面完成预订后，可以简单地在Apple日历、Google日历和Outlook（2003和2007）中追加预订内容。

请注意：

  • 不能从智能手机追加预订内容。请使用ANA网站。
  • 此功能不可用于没有预订的不定期机票或者通过旅行社预订的其他航空公司的机票。根据预订情况的种类，航班行程变更后，可能无法下载整个旅程。
  • 搭乘航班信息还会显示候补预约或等待其他公司回复的航班。请注意，这并不代表有确定座位。
  • 对于在日本出发日前一天到达目的地的航班，日历预订内容中的“开始时间”、“结束时间”均显示为出发时间。
  • 预订变更时，或已添加到日历上的搭乘航班信发生变更时，不会自动更新。请从“预订确认”页面下载新的预订信息。
  • 关于日历的操作方法和设定的自定义方法的详细说明和使用方法，请咨询日历APP的提供商。

出票

Apple 钱包

使用“Apple 钱包”APP时，可以使用iPhone（iOS 9.0以上）搭乘ANA的国内航线和国际航线。在自己的“Apple 钱包”内的Pass行证或在会员卡中保存登机牌，旅行会更加便利。

请注意：
  • 添加了登机牌的乘客，可以从Apple 钱包 APP收到关于预订航班的变更、取消、延误等通知。
  • 使用安装了iOS 9.0以上的OS的终端的乘客，可以使用“Apple 钱包”APP。
  • 使用安装了iOS 6.0到8.4.1的终端时，可以使用“Passbook”。
  • “Apple 钱包”在iPad上不能使用。
  • “Apple 钱包”在Andriod™终端上不能使用。
“Apple 钱包”的使用方法
第1步：从值机状态的页面登录“Apple 钱包”。
  • 完成在线值机（国际航线的24小时内可用）后，选择“添加到Apple 钱包”。
  • 每次搭乘的航班都可以从“添加到Apple 钱包”页面登录。
  • 请注意：“Apple 钱包”在亚洲和大洋洲的一部分机场不能使用。
  • 其他航空公司的登机牌，请于出发日当天在ANA承运航班的出发机场领取。
  • 关于步骤的详细说明，以及可使用电子登机牌的机场列表，请查看“在线值机”页面。
第2步：在APP中登录Pass。
  • 点击登机牌右上角的“添加”，完成登录。
  • 在“Apple 钱包”中生成ANA的Pass。请打开APP查看。
第3步：在出发日显示Pass。
  • 出发时，TSA的工作人员会扫描智能手机的二维码。
  • 如要查看Pass的背面或乘机信息，请点击右上角的“・・・”图标。
  • 在Pass的背面还可以确认要进入的休息室。
航班情况的通知
  • 航班取消将通过Pass通知乘客。
  • 看到通知信息后打开Pass，便会出现圆形标记，显示最新的信息。
    • 请注意：只有从通知信息访问Pass时，才会出现圆形标记。
  • 仅适用于从“Apple 钱包”登录的ANA国际航线。
    • 使用国际航班的在线值机进行值机的日本国内航班，不在“Apple 钱包”通知范围内。
  • 登机时间发生变更时，会显示新的登机时间，并出现圆形标记。
使用条件
  • 如果使用不支持的终端，在领取登机牌的页面不会显示“添加到Apple 钱包”的按钮。此时请选择其他选项。
  • 搭乘的乘客选择“通过手机终端领取”，可直接从智能手机领取二维码形式的电子登机牌。
  • 在航班的每个航段都需要登录Pass。
    • 如果超过2个航段，请在各航段分别登录。
  • 航班情况通知的送达仅限于连接网络的情况。
    • 请注意：根据接收和连接的情况，有时通知无法送达乘客的终端。
  • 如果取得Pass后预订改签（包括变更座位指定），该变更不会自动更新。如要更新Pass，需要再次添加Pass。
  • 登机后，Pass不会自动删除。请手动删除Pass。

  • iPhone、Apple Watch、iPod touch以及Wallet/Passbook是美国及其他国家的Apple公司的注册商标。

机票和电子机票

  • 机票是指由航空公司或其指定代理店出票的“电子客票行程单”或电子乘机联。
  • 伴随机票的购买，各航空公司就与乘客签订了运输合同。
  • 乘客的机票信息将作为ANA乘机联保管，其复印件以“电子客票行程单”的形式交付给乘客。
    • 乘客在办理值机或者入境审查时可能会用到行程单。
电子客票行程单
  • 记载了适用票价、机票信息以及旅程的A4纸大小的行程单。
  • ANA出票的机票号码是以205开头的13位数字。
  • 下图是电子客票行程单的样本。
机票规则
  • 机票只对票面上显示姓名的乘客有效。不可变更姓名。也不可转让他人。
  • 请准备好您乘坐航班的“电子客票行程单”。
  • 如果以通常价格购买机票，则在最初航段出发日（如果未使用，则为出票日）起1年之内有效。
    • 如果是特价机票，有效期限不同，适用期限也不同。
  • 请您根据机票中所记载的从出发地开始的旅程按顺序使用机票。
    • 如果您有未使用的航段，那么机票和剩余的航段也将失效。
  • 出票手续费：通过致电客服人员，或在ANA机场柜台新出票的国际航线机票，原则上每1张机票都要收取一定的出票手续费。
    • 可能1张机票包括多个航段。

电子收费凭证

  • EMD（Electronic Miscellaneous Document）是由航空公司或指定的代理店所发行的电子凭证。
  • EMD票面上所记金额具有其价值，作为票面所记载的用途的兑换券，或是作为手续费的凭证。
  • 由于所使用的支付货币的限制，在您购票的国家以外的地区可能无法使用EMD。
EMD乘客凭证（PASSENGER RECEIPT）
  • 记载了费用信息、发行用途的A4纸大小的行程单。
  • 改签或退票时，请在咨询时告知票面左上方的“EMD号码”。
  • ANA发行的EMD是以205开头的13位数字。
  • 下图是EMD乘客凭证的样本，仅供参考。
EMD规则
  • 预订改签手续费：根据预订改签，EMD可作为收取变更手续费的凭证。
  • Residual Value for Refundable Balance：EMD中显示由于旅程变更而退款的金额。如果您是通过旅行社购买机票，请咨询您购票的旅行社。
  • Upgrade:可作为用于升舱所需里程数或升舱积分扣减的乘客凭证。
  • 付费贵宾休息室服务:可作为使用付费贵宾室服务时支付费用的凭证。
  • 购买其他服务时也可发行EMD。
使用示例
  • EMD只对票面上显示姓名的乘客有效。不可变更姓名。也不可转让他人。
  • 使用相应服务时，请准备好EMD乘客凭证。
  • EMD自发行之日起1年内有效。
  • EMD的发行不需要出票手续费。

旅行途中的地面交通服务

ANA为您的旅行准备了舒适快捷的地面交通工具。

珠江三角洲渡轮转乘服务

通过渡轮转乘服务，无需通过香港的入境审查及海关检查，就可以从珠江三角洲各城市转机到日本。豪华经济舱和公务舱的乘客可以乘坐渡轮。

欧洲地区的铁路交通服务

搭乘日本、欧洲航段的乘客，可以购买连通欧洲地区的铁路区段的车票。也适用德国铁路和InterCity铁路服务。头等舱、公务舱、豪华经济舱和经济舱的乘客，可以使用此铁路服务。

座位电源、USB接口

仅特定航线和机型可使用。详情请查看“座位一览”页面。
除飞机起飞降落、地面滑行以及停场时，其他时间均可使用。
请注意：即使在可使用时间内，我们也可能会停止供电，恕不另行通知。

插座的使用方法

使用笔记本电脑
  • 在使用笔记本电脑之前，请确认电脑电源为110V/60Hz，最大功率为60W。
  • 确认机型可使用后，请先将AC适配器的插头插入您的电脑。
  • 然后，将AC适配器的插头插入扶手附近的插座最里端。
    • 如果没插入到位，安全装置将会运行，可能不会供电。
    • 如果您的电脑电源插头无法插入，我们可为您提供延长线。（请注意：仅限波音777-300ER的公务舱可使用延长线。）
使用其他电子设备
  • 在使用其他电子设备之前，请确认电源为110V/60Hz，最大功率为60 W。
  • 可使用的设备：
    • 智能手机、平板电脑、数码音频设备、摄像机、数码相机、电子游戏机等。
  • 插头的形状要能牢牢地插入插座，不需要适配器。
  • 如果您的设备符合上述条件，请打开开关。
  • 然后，将AC适配器的插头快速插入扶手附近的插座最里端。
  • 请注意：睡眠呼吸暂停综合征的乘客，如要使用CPAP（持续正压通气呼吸装置），请浏览“其他需要帮助的乘客”页面。

插座的详情

  • 插座可使用以下3种类型的插头。2孔的欧标和2孔或3孔的美标。
  • 不可使用适配器。使用适配器后安全装置可能会运行，会有造成短路的风险。
  • 如您的笔记本电脑使用英标插头，我们将提供2种机内使用的专用接线。
    • 需要使用时，请告知客舱乘务员。接线数量有限。
  • 根部配备绝缘套管的插头无法牢牢地插入插座最里端，因此不可使用。
    • 如果您的设备使用此类型的插头，我们可以提供机内使用的专用接线。但因数量有限，可能无法满足您的需求。敬请谅解。

安全注意事项

  • 虽然座位的最大输出功率可达到60W，但是根据其他旅客的使用状况，也可能出现无法给设备充电的情况。
  • 为保证旅客安全，我们对各座位的最大供电量进行了限度设置。
  • 如您在使用时超过限度，邻座旅客的供电也会被切断。
  • 机内座位配备的USB接口，有供给电流值为1.5A以上的机型以及0.5A的机型，两者均可用于充电。
    但是，根据旅客使用的设备，可能因供电电流不足，而出现无法充电的情况，或者无法显示“充电中”的提示。
  • 如系统发生故障或供电超标时，我们可能会切断电源，恕不另行通知。
  • 此时若使用笔记本电脑，可能会导致数据丢失，推荐自行备份数据。
  • 若边连接着插座边使用设备，部分触摸式设备可能会反应迟钝，或反应错误。（即使发生这类情况，设备也不会受损。）
  • 插座的充电功能也不会受到影响。
  • 请勿使用毛毯等盖住电脑的通风孔。
  • 请使用无故障的电源线和设备。
  • 客舱乘务员无法判断您的电子设备是否符合上述注意事项。
  • ANA对因使用电源带来的设备损坏等恕不负责。

LED灯

座位的插座中有LED灯，通过在插座下面或旁边的LED灯显示电源是否可用。

在插座有2色灯的座位，绿色表示电源可以使用，红色表示暂时不能使用或完全不能使用。在插座有3色灯的座位，绿色表示可以使用，橙色表示暂时不能使用，红色表示完全不能使用。

如果红色或橙色的灯亮起，请进行如下操作。

  • 请拔下插头，在确认LED灯变为绿色后，再次插入AC适配器。

以下原因可能导致电力不足。

  • 连接插座的设备耗电量达到75W以上。
  • 超过可供给的供电量。
  • AC适配器未牢牢地插入最里端。

如果红色或橙色的灯亮起，并非系统故障。
